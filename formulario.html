Estrutura do Formulário
Tags Descrição Código Fonte 
<form>   </form>
 Dentro dessa tag será inserido todo conteudo do formulário. 
<input type="" name=""/>
 Essa tag sinaliza entrada de dados. Ela vem acompanhada do parametro type que indica o tipo da entrada e do parametro name que é o nome do entrada.

Tipos de Entradas
Types Descrição Exemplos Código Fonte
type="text" Entrada do tipo caixa de texto. 
<input type="text" name="texto"/>
type="password" Entrada do tipo caixa de senha. 
<input type="password" name="senha"/>
type="checkbox" Criação de caixas de seleção. 
<input type="checkbox" name="caixa"/>
<input type="checkbox" name="caixa"/>
type="radio" Esse tipo é semelhante ao anterior com a diferença é que com o radio só é possivel selecionar uma única opção. 
<input type="radio" name="opção"/>
<input type="radio" name="opção"/>
type="submit" Cria um botão que submete os dados do formulário. 
<input type="submit" value="submição"/>

Tags Descrição Exemplos Código Fonte

 Cria o botão expansivel que guarda as opções. 
<option>   <option/>
 Insere as opções dentro do select.

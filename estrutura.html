Tags Descrição Código Fonte 
<!DOCTYPE html>
 Essa tag identifica o tipo de documento para o browser. 
<html>   </html>
 Tag principal, dentro dela que ficrá o programa que se deseja desenvolver. 
<head>   </head>
 Tag que contém informações técnicas importantes não exibidas para o usuário final. 
<body>   </body>
 Dentro de desta Tag é inserido as informações exibidas para o usuário final.
